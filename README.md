# sk1razze-shop-bot

Магазин цифровых товаров с системой франшиз. Делал для клиента - хотел полноценный движок под несколько магазинов с одной панелью управления.

## Что умеет

- продажа цифровых товаров (текстовые позиции + файлы)
- два способа оплаты: CryptoBot и Lolz.market
- система франшиз — один бот запускает дочерние боты для партнёров, у каждого своя витрина и заказы
- приватные каналы с подписками — доступ выдаётся автоматически, при истечении кик
- реферальная система с прокси-счётом
- промокоды с лимитом использований и скидкой
- вывод средств с подтверждением от владельца
- рассылка по сегментам (все / купившие / не купившие)
- полная админка: категории, товары, пополнение позиций, баннер, тексты, статистика

## Стек

aiogram 3, aiosqlite, aiohttp, python-dotenv

## Запуск

```
pip install -r requirements.txt
cp .env.example .env
# заполнить .env
python bot.py
```

## Структура

```
handlers/
├── admin/       — управление магазином
├── franchise/   — панель франчайзи
├── shop/        — каталог и покупки
├── payment/     — пополнение (CryptoBot, Lolz)
└── profile/     — профиль, рефералы, промокоды

services/
├── bot_manager.py        — динамическая загрузка франчайзи-ботов
├── delivery.py           — выдача товара после оплаты
├── subscription_manager.py — проверка и кик из приватных каналов

db/
├── models.py     — схема БД
└── queries/      — SQL-запросы по модулям
```
